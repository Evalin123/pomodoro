# 🍅 Pomodoro

A basic Pomodoro timer web app built with HTML, CSS, and JavaScript.

## Features

- **25-minute** focus session (Pomodoro)
- **5-minute** short break
- **15-minute** long break (auto-triggered every 4 Pomodoros)
- Animated circular progress ring
- Start / Pause / Reset / Skip controls
- Keyboard shortcuts: `Space` to start/pause, `R` to reset
- Session counter with 4-dot progress tracker
- Toast notifications on session completion
- Audio beep via Web Audio API
- Color theme per mode (red / teal / blue)

## Usage

Open `index.html` in any modern web browser — no build step required.
